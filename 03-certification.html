3.1 EMR 인증제 개요와 2주기 기준 변화

한국의 EMR 인증제는 의료법 제23조의2 + 시행령 제10조의7~9에 근거한 자율 인증입니다. 미인증 시 시스템 운영 자체는 가능하지만 인증 표시 사용이 금지되며, 실제로는 진료정보교류·공공 연계·대형기관 도입에서 사실상 큰 제약을 받습니다.

  • 1주기 (2020.7~2024) — 90개 인증 항목. 기능성 중심 평가
  • 2주기 (2025.1 시행~) — 59개 항목으로 통합 간소화. 보안성·상호운용성 기준 강화
  • 관할 — 보건복지부 + 한국보건의료정보원(KHIS) 운영
  • 실무 절차 — 사전점검 → 신청 → 사전 문서심사 → 현장심사 → 보완조치 → 인증위원회 심의
  • 준비 기간 — 컨설팅 포함 6~12개월. KHIS 컨설팅 프로그램 활용 가능
  • 인증대상 범위 주의 — KHIS 인증기준상 치과·한방·정신·요양병원은 인증대상에서 제외됩니다. 본 보고서가 진료과 카테고리 리더로 인용한 덴트웹·하이덴(치과), 트루닥 멘탈(정신과)은 인증제 트랙이 다르므로, 의원급 EMR 인증과 같은 표에서 비교할 때 이 차이를 전제로 해석해야 합니다 — KHIS 인증기준 페이지

3.2 기능성 · 상호운용성 · 보안성 요구사항

2주기 59개 항목은 세 영역으로 나뉩니다. 각 영역은 단순 체크리스트가 아니라 EMR이 의원 운영의 백본으로 기능할 수 있는지를 검증하는 평가입니다.

3.3 ISMS-P / CSAP / 청구 SW 보안인증

보안 영역은 EMR 인증제 외에 별도 트랙의 인증이 누적됩니다. 클라우드 모델을 채택하면 이들 보유가 사실상 진입조건이 됩니다.

  • ISMS-P (정보보호 및 개인정보보호 관리체계 인증) — 보유 시 EMR 인증제 보안성 영역 일부 면제
  • CSAP (클라우드보안인증) — 공공 영역 및 일부 의료기관 도입 시 요구
  • 심평원 청구 SW 보안인증 — 청구 SW가 별도 갖춰야 하는 보안 요건
  • 의료기기 SW(SaMD) 인허가 — AI 보조 판독·예측 기능을 의료 의사결정에 직접 연결할 경우 식약처 의료기기 인허가 트랙으로 진입

3.4 규제가 신규 진입자에게 주는 의미

규제는 신규 진입자에게 두 방향으로 작용합니다.

⚠️
진입 비용 — '제도적 진입 비용'은 누구에게나 동일
59개 인증 항목, 6~12개월 준비기간, ISMS-P/CSAP/청구 SW 보안인증 — 일반 EMR이든 진료과 특화 EMR이든 동일하게 부과됩니다. 도메인 특화 자산(임상 데이터·AI·네트워크)은 차별화는 만들어 주지만 제도 비용을 줄여주지는 않습니다.
🛡️
진입 장벽 — 영세 진입자를 자연 도태시키는 필터
동시에 이 규제는 '영세 EMR 업체가 지속적인 수가 고시·인증 업데이트 부담을 못 견뎌 폐지하는 구조'의 원인이기도 합니다. 즉 신규 진입자에게는 비용이지만, 일정 규모 이상의 진입자에게는 경쟁자 자연 정리 효과로 작용합니다.
함의: 자체 EMR을 추진하든, 외부 EMR을 도입하든, '인증·청구·보안 비용은 어떤 경우에도 누군가는 지불한다'는 사실이 출발점입니다. 이 비용은 8장 현실적 난이도에서 다시 다룹니다.
